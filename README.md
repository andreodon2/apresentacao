# Instruções

## Projeto de currículo

Este projeto foi idealizado no passado para eu praticar HTML, CSS, e recentemente decidi praticar React/Vite e portanto estou refatorando o código para este framework.

## Instalando o Vite
Após você ter feito o fork e o clone do projeto em seu dispositivo, é necessário a instalação das dependências do Vite. Para isso execute o comando dentro da pasta do projeto (veja como acessar o projeto):
- npm install

## Acessar o projeto

Para acessar o projeto você deve acessar o caminho:
- Vite/apresentacao

Depois rodar o comando:
- npm run dev

O terminal do VS Code irá gerar um endereço de localhost onde o projeto estará ativo para visualização do front-end.
